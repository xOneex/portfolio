Devils Crown

Personal Project


Devils Crown is a boss-focused action game with a Souls-like atmosphere. developed using MonoGame and written in C#. It takes inspiration from titles like Dead Cells and Hades, while its mechanics are influenced by the Dark Souls series.

The core concept of the game revolves around defeating the main boss as quickly as possible, emphasizing skill and strategy, obtaining achievements along the way. This project was designed to showcase my programming abilities and my capacity to create a polished and functional game from scratch without an engine [also it has controller support].

All the visuals were created in Blender as 3D models, which were then rendered into pixel art for easier animation and a unique aesthetic.

Apart from the sound and music, every aspect of the game was created solely by me.

VOVE
Personal Project


VOVE is a rhythm-based game inspired by osu!, developed using Processing and written in Java. Players navigate through levels synced to music, requiring precision and timing to achieve high scores.

The game features a fully designed UI and menu system, and a polished visual aesthetic and a custom undertale-themed level, built entirely from scratch without an engine.

Apart from the sound, music and background art, every aspect of the game was created solely by me.

Also, this game was actually created for one of my University modules.

Dead-Sector [IN DEVELOPMENT]
Personal Project


Dead-Sector is a multiplayer 3D atmospheric game developed in Unity and written in C#. Set in a dark, futuristic, industrial world, players take on the role of experimental recovery units on surface salvage operations.

The game features pseudo-random map generation, procedural animation, a reactive environment, and peer-to-peer multiplayer through Steam using the host's machine as a dedicated server. All built with custom 3D assets and atmospheric lighting.

The world is backed by an extensive lore system still in active development, with a large cast of voiced characters and an original soundtrack produced by a trusted music producer.

Every aspect of the game [excluding the soundtrack] is being created solely by me.

Nova [IN TESTING]
Personal Project


Nova is a Discord-integrated AI written in C#, using a 12B parameter LLM, designed to feel less like a bot and more like a person. Built with sentiment analysis and realistic emotional modelling, Nova responds dynamically based on the tone, context of conversations and how she feels [nova is able to evolve/overwrite parts of her own personality and emotions].

Nova maintains a persistent memory system and a cross-referenced social memory system that tracks and builds relationships between users across a server. She remembers people, forms opinions, and will gossip to others based on what she knows and how she feels. She's also able to jump into any conversation she chooses.

Nova runs as a live deployed system across multiple Discord servers, not just something that sits on a machine.

As of right now, Nova is only permitted to join set, restricted servers for testing purposes.

Every aspect of Nova including her personality system, memory architecture and emotional modelling was designed and built solely by me.
