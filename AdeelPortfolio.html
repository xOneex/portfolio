Devils Crown

Personal Project


Devils Crown is a boss-focused action game with
a Souls-like atmosphere. developed using MonoGame and
written in C#. It takes inspiration from titles like Dead
Cells and Hades, while its mechanics are influenced by
the Dark Souls series.

The core concept of the game revolves around defeating
the main boss as quickly as possible, emphasizing skill
and strategy. This project was designed to showcase
my programming abilities and my capacity to create a
polished and functional game from scratch.

All the visuals were created in Blender as 3D models,
which were then rendered into pixel art for easier
animation and a unique aesthetic.

Apart from the sound and music, every aspect of the
game was created solely by me.
